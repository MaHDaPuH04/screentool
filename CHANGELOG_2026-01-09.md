# ОТЧЕТ ОБ ИЗМЕНЕНИЯХ - Auto Screenshot Tool

**Версия:** 1.1.0  
**Дата:** 09.01.2026 
**Исполнитель:** [Ваше ФИО]  
**Статус:** Внедрено 

## ЦЕЛИ ИЗМЕНЕНИЙ
1. Улучшение UX работы с превью скриншотов
2. Упрощение структуры групп для отчетности
3. Защита от случайной потери данных
4. Улучшение навигации по скриншотам

## ИЗМЕНЕННЫЕ ФАЙЛЫ

### 1. preview_dialog.py - УЛУЧШЕНИЕ ПРЕВЬЮ
**Добавлено:**
- Навигация стрелками клавиатуры (← →)
- Счетчик скриншотов "X/Y" в заголовке
- Автоматическое обновление после удаления
- Поддержка полноэкранного режима (F11)

**Изменено:**
- Убраны кнопки навигации из интерфейса
- Добавлены стандартные кнопки окна (максимизация, закрытие)
- Улучшено масштабирование изображений
- Создаем листы только при наличии скриншотов:
    Проверяем if screenshots: перед созданием листа
- 1я попытка устранения бага "неразворачивания окна"

### 2. shot.py - НОВАЯ СИСТЕМА ГРУПП
**Добавлено:**
- Предопределенные имена групп:
  - poll+calib (1)
  - TIP (2)  
  - ver (3)
  - TM (4)
  - PDT (5)
  - press_1, press_2... (6+)
- Автоматическое определение текущей группы
- Оповещение превью об удалении скриншотов

### 3. window.py - ЗАЩИТА ОТ ПОТЕРИ ДАННЫХ
**Добавлено:**
- Блокировка кнопки "Очистить папку" до экспорта
- Автоматическое обновление метки текущей группы
- Визуальные индикаторы состояния

### 4. excelexport.py - ОБНОВЛЕНИЕ ЭКСПОРТА
**Исправлено:**
- Поддержка новых имен групп при экспорте в Excel
- Правильная сортировка листов в файле
- Обработка пустых папок

## НОВЫЕ ВОЗМОЖНОСТИ

### Для пользователя:
1. **Удобная навигация** - стрелками по скриншотам
2. **Защита данных** - нельзя очистить до экспорта
3. **Логичные имена** - poll+calib, TIP, ver, TM, PDT
4. **Автоматическое обновление** превью после удаления

## ИСПРАВЛЕННЫЕ ОШИБКИ
1. Циклический импорт между shot.py и preview_dialog.py
2. Превью не обновлялось после удаления скриншотов
3. Некорректный счетчик при удалении
